# 🦋 NAVI Frontend

소아암 환우와 보호자를 위한 심리사회적 지지 커뮤니티 플랫폼

## 기술 스택

- **Framework**: Next.js 14 + TypeScript
- **Styling**: Tailwind CSS
- **State**: Zustand
- **API**: Axios

## 주요 기능

- **하루공유**: 매일 기분 이모지 + 세줄일기 기록
- **공감하기**: 6종 이모지 반응
- **팔로우**: 관심 유저 팔로우/피드 구독

## 🤖 AI로 구현하기

이 프로젝트는 **Claude AI**를 활용하여 개발했습니다.

- UI/UX 설계 및 컴포넌트 구조화
- Next.js 페이지 및 컴포넌트 구현
- 코드 리뷰 및 리팩토링

## 실행 방법

```bash
# 패키지 설치
npm install

# 환경변수 설정
cp .env.example .env.local

# 개발 서버 실행
npm run dev
```
